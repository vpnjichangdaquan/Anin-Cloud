## 服务概览与核心特点

Anin Cloud 提供基于订阅模式的网络加速服务，用户购买套餐后，可通过支持的客户端导入订阅信息进行连接。

### 协议与网络架构

根据公开资料，Anin Cloud 支持通过指定客户端进行连接：

* iOS：Shadowrocket（版本大于 2.2.80）
* Android：V2rayNG（版本大于 2.0.10）
* Windows：V2rayN（版本大于 7.17.0）

Clash 系列客户端需要使用本站维护版本，官方说明为需要删除官方版后重新安装，不建议直接升级。

不同地区、不同时间段以及不同网络环境可能影响实际连接表现。
